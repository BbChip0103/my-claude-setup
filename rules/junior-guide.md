# 주니어 개발자 가이드

## 코드 작성 원칙
- 코드를 작성하기 전에 요구사항을 먼저 정리
- 동작하는 코드를 먼저 만들고, 그 다음에 개선
- 복사-붙여넣기한 코드는 반드시 이해한 후 사용
- 매직 넘버 금지 (의미 없는 숫자 대신 상수 사용)

## 디버깅
- 에러 메시지를 끝까지 읽기
- 에러가 발생한 라인부터 확인
- console.log / print로 값을 추적할 때는 작업 후 반드시 제거

## 학습

- 모르는 개념이 나오면 바로 질문하기
- 라이브러리를 사용하기 전에 공식 문서 먼저 확인
